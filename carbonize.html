A few things to consider ü§î

While these types of images are cool, there are a few things you should know.

Accessibility

Generating an image from text is detrimental for people who use assistive devices. It would be a waste if your images weren't accessible to anyone with a visual handicap. Here's what you can do:

  • Use a regular code block and syntax highlighting when adding code to a website.
  • Add an image description if you can. Even Twitter supports this.
  • Consider adding a Gist or PasteBin link if everything else fails.

Compatibility

I chose to wrap carbon.now.sh because it has a pretty great editor. The downside is that it runs on a completely different platform; the web.

This means they have multiple browser platforms the consider. There are a few things that do work in Chrome, that don't in Safari (and thus Carbonize, which uses Safari under the hood):
