Reign

The Supreme Remote for Spotify

Version 1.0.4 | For Mac OS X 10.6+ | FREE

The Spotify remote for friends, colleagues, housemates and yourself that works in any browser.

Control Spotify from any device with a browser!

  • Phones
  • Tables
  • Other computers
  • Playstation Vita
  • Playstation 3
  • Web enabled TVs
  • E-Readers (yes, really)
  • Anything else with a browser

If you think of something that’s not on this list and it works, let me know!

Easily share your remote

By using Safari’s Bonjour Bookmarks/Favorites anyone can discover your remote. Even without having Reign installed!

Read the guide to set this up.

Customize the experience

Use themes to personalize your Spotify remote. Select a background image, or completely change the HTML & CSS.

You can even take it a step further and create a native custom client. Check the developer page for more information.

Alfred 2 Workflow

Some of you might be familiar with Alfred. It’s a nifty tool that aims to increase your productivity by executing many tasks using keyboard shortcuts and keywords.

Reign now has it’s own Alfred 2 workflow that helps you to remote control Spotify clients super fast, using your keyboard. First download Alfred, then get the workflow and start typing.

Like Reign, the workflow is free and open-source. Contribute on GitHub.
