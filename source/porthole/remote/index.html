Toggle AirPlay speakers on or off and change the volume from the comfort of your couch by using Porthole's remote control.

Together with Unified Remote you can control Porthole from anywhere. Unified Remote works with iOS, Android and Windows Phone.

The remote works with every release of Porthole since version 1.7.0!

Getting & Installing Unified Remote

Mac OS X

Download the server application and install it on the Mac you run Porthole on.

Mobile

Get the app for your mobile platform:

  • iPhone or iPad
  • Android
  • Windows Phone

Enabling The Porthole Remote

Unified Remote includes a few built-in remotes to control your Mac. We'll need a specific one to control Porthole. Follow these steps to get it.

Note that Unified Remote is not made by Danger Cove. For support contact Unified Remote directly.

On the main menu select the option to get "More Features".

purchase an additional remote from the overview to be able to activate the remote for porthole.

optionally you can unlock the full version which will let you enable an unlimited amount of extra remotes and enables other additional features.

On the main menu click the indicated button to add a new remote.

Tap on the Porthole remote in the overview.

Go back to the main menu and select the Porthole remote from the list.

Wait for your speakers to appear and you're ready to control Porthole from the comfort of your couch!

Troubleshooting

Note that Unified Remote is not made by Danger Cove. For support contact Unified Remote directly.

Make sure both the Mac and mobile device are connected to the same WiFi network.

You can manually select the server to use by tapping the 'option' button in the top left corner, then clicking servers and selecting the correct server from the overview.
